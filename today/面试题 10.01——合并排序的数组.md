# 题目
给定两个排序后的数组 A 和 B，其中 A 的末端有足够的缓冲空间容纳 B。 编写一个方法，将 B 合并入 A 并排序。

初始化 A 和 B 的元素数量分别为 m 和 n。

示例:

输入:
A = [1,2,3,0,0,0], m = 3
B = [2,5,6],       n = 3

输出: [1,2,2,3,5,6]

说明:

    A.length == n + m

* 代码：
    ```C++
    class Solution {
    public:
        void merge(vector<int>& A, int m, vector<int>& B, int n) {
            int last = m+n-1;
            while(m>0&&n>0)
            {
                if(A[m-1]>B[n-1])
                {
                    A[last] = A[m-1];
                    --m;
                }
                else
                {
                    A[last] = B[n-1];
                    --n;
                }
                --last;
            }
            while(n>0)
            {
                A[last] = B[n-1];
                --n;
                --last;
            }
        }
    };
    ```
