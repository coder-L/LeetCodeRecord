# 题目
给定一个正整数 n，将其拆分为至少两个正整数的和，并使这些整数的乘积最大化。 返回你可以获得的最大乘积。

* 示例:

>输入: 2<br>
输出: 1<br>
解释: 2 = 1 + 1, 1 × 1 = 1。

>输入: 10<br>
输出: 36<br>
解释: 10 = 3 + 3 + 4, 3 × 3 × 4 = 36。

* 思路：有O(n)的解法，待完善

* 代码：
    ```C++
    class Solution {
    public:
        int integerBreak(int n) {
            vector<int> dp(n+1,0);
            dp[1]=1;
            for(int m=2;m<=n;++m)
            {
                dp[m] = m;
                for(int k=1;k<m;++k)
                {
                    dp[m] = max(dp[m],k*dp[m-k]);
                }
            }
            int result = 1;
            for(int m=1;m<n;++m)
            {
                result = max(result,dp[m]*dp[n-m]);
            }
            return result;
        }
    };
    ```
